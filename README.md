# project_boulder_fastapi

1. 가상화

    1.1 가상화 만들기
    ```
    $ make env
    ```
    1.2 필요 라이브러리
    ```
    $ make setup
    ```
    1.3 가상화 실행
    ```
    $ conda activate fastapi
    ```
    1.4 실행
    ```
    $ python3 main.py 
    ```

2. ORM (sqlalchemy)
    
    1.1 ORM 관련 라이브러리 설치
    ```
    $ pip install sqlalchemy
    ```
